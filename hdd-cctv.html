¿Qué disco duro usar para cámaras de seguridad?

El tamaño del disco depende de la cantidad de cámaras, la calidad de grabación y los días de almacenamiento. Esta calculadora permite estimar qué HDD necesitas para un sistema CCTV.

Para sistemas de videovigilancia se recomienda usar discos diseñados para grabación continua como los WD Purple o Seagate SkyHawk.
